# Sales Data Analysis Project

## Overview
This project analyzes sales data using SQL and Power BI to extract insights about revenue, profit, and product performance.

## Tools Used
- SQL (PostgreSQL)
- Power BI
- Python (optional for preprocessing)

## Key Analysis
- Total revenue and profit analysis
- Revenue by country
- Revenue trend over time
- Top 3 product categories by revenue

## Dashboard
The Power BI dashboard includes:
- KPI metrics (Revenue, Profit)
- Revenue by country
- Revenue trend
- Top categories analysis
- Interactive filters

## Key Insights
- Revenue is concentrated in key countries
- Business shows growth over time
- Top categories contribute most of the revenue

## Files
- `data/`: dataset used
- `sql/`: SQL queries
- `dashboard/`: Power BI file
